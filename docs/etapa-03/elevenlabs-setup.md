# Etapa 03 - Setup ElevenLabs

## Criar chave

1. Acesse [ElevenLabs](https://elevenlabs.io).
2. Crie uma conta ou faça login.
3. Abra o perfil da conta.
4. Copie a API key.

## Configurar `.env`

```text
ELEVENLABS_API_KEY=sua_chave_elevenlabs_aqui
ELEVENLABS_VOICE_ID=5p9IbzcK4R8rN1fpGdMF
ELEVENLABS_MODEL=eleven_multilingual_v2
```

## Observações

- O `VOICE_ID` pode ser trocado por outra voz disponível na sua conta.
- O modelo `eleven_multilingual_v2` é adequado para português.
